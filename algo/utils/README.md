# 累積和・いもす法・Fenwick Tree の使い分け

| 手法 | 向いている状況 | 計算量 |
| --- | --- | --- |
| 累積和 | 配列を更新せず、区間和を何度も求める | 構築 `O(N)`、区間和 `O(1)` |
| いもす法 | 区間加算をすべて行った後、最後にまとめて値を求める | 区間加算 `O(1)`、復元 `O(N)` |
| Fenwick Tree | 値の更新と累積和・区間和の取得が交互に行われる | 更新・取得ともに `O(log N)` |

## 判断基準

- 更新がない: 累積和
- 区間更新を最後にまとめて反映できる: いもす法
- 更新の途中で問い合わせに答える必要がある: Fenwick Tree

いもす法で区間 `[L, R]` に `x` を加える場合は、差分配列に次のように記録し、最後に累積和を取る。

```python
diff[L] += x
diff[R + 1] -= x
```

Fenwick Treeは基本的に一点更新と累積和の取得を扱う。差分配列をFenwick Treeで管理すれば、区間加算と一点取得もオンラインで処理できる。
